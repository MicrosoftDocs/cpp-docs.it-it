# <a name="operator--"></a>Operatore -

Restituisce *expression1* meno *expression2*. Il secondo operatore inverte il segno dell' *espressione*.

## <a name="syntax"></a>Sintassi

> *expression1* **-** *expression2*
>
> __-__*espressione*
