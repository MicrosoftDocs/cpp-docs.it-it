# <a name="operator-sign"></a>Operatore SIGN?

Stato del flag di firma. Utilizzato solo all'interno di [. SE](dot-if.md), [. WHILE](dot-while.md)o [. Ripetere](dot-repeat.md) i blocchi e valutati in fase di esecuzione, non in fase di assembly.
