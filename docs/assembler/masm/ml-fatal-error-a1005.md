# <a name="ml-fatal-error-a1005"></a>Errore ML irreversibile A1005

**limite assembler: tabella con nome parametro della macro completa**

È stato definito un numero eccessivo di parametri, variabili locali o etichette macro per una macro. La tabella dei nomi delle macro non contiene più spazio.

Definire nomi più brevi o minori oppure rimuovere le macro non necessarie.
