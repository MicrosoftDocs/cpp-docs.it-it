# <a name="operator-highword"></a>Operatore HIGHWORD

Restituisce i 16 bit alti dei 32 bit inferiori dell' *espressione*. Le espressioni MASM sono valori a 64 bit.

## <a name="syntax"></a>Sintassi

>  *Espressione* HIGHWORD
