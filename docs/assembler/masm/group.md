# <a name="group"></a>GROUP

(solo MASM a 32 bit). Aggiungere i *segmenti* specificati al gruppo denominato *nome*.

## <a name="syntax"></a>Sintassi

>  ⟦ *segmento* **gruppo** nome __,__ *segmento* ... ⟧
