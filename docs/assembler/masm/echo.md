# <a name="echo"></a>ECHO

Visualizza il *messaggio* nel dispositivo di output standard (per impostazione predefinita, la schermata).

## <a name="syntax"></a>Sintassi

>  *Messaggio* Echo

## <a name="remarks"></a>Commenti

Uguale a [% out](percent-out.md).
