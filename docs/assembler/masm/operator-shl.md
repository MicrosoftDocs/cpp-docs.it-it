# <a name="operator-shl"></a>Operatore SHL

Restituisce il risultato dello spostamento dei bit del numero di bit *a sinistra dell'* *espressione* .

## <a name="syntax"></a>Sintassi

> *conteggio* delle *espressioni* **SHL**
