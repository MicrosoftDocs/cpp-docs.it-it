# <a name="listif"></a>.LISTIF

Avvia l'elenco di istruzioni in blocchi condizionali false.

## <a name="syntax"></a>Sintassi

> **.LISTIF**

## <a name="remarks"></a>Commenti

Uguale a [. LFCOND](dot-lfcond.md).
