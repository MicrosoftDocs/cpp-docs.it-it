# <a name="typedef"></a>TYPEDEF

Definisce un nuovo tipo o **proto** denominato *Name*, equivalente a *Type* o *protoDefinition*.

## <a name="syntax"></a>Sintassi

> *nome* **typedef** *tipo*  |  **proto** *protoDefinition*
