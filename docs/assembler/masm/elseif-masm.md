# <a name="elseif"></a>ELSEIF

Combina [else](else-masm.md) e [if](if-masm.md) in un'unica istruzione.

## <a name="syntax"></a>Sintassi

>  *ConstantExpression* ElseIf\
> *istruzioni*\
> ⟦**Else**\
> *else-istruzioni*⟧ \
> **ENDIF**
