# <a name="sword"></a>SWORD

Alloca ed eventualmente Inizializza una parola firmata (2 byte) di archiviazione per ogni *inizializzatore*. Può essere usato anche come identificatore di tipo in qualsiasi punto in cui un tipo è valido.

## <a name="syntax"></a>Sintassi

> *nome*  *inizializzatore* spada ⟦__,__ *inizializzatore* ... ⟧
