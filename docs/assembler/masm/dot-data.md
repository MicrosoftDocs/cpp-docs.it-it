# <a name="data"></a>.DATA

 (solo MASM a 32 bit). Se usato con [. MODELLO](dot-model.md), avvia un segmento di dati near per i dati inizializzati (nome del segmento _DATA).

## <a name="syntax"></a>Sintassi

> **. DATI**\
> ⟦ *segmentItem* ⟧...
