# <a name="end"></a>FINE

Contrassegna la fine di un modulo e, facoltativamente, imposta il punto di ingresso del programma su *procId*.

## <a name="syntax"></a>Sintassi

> **Fine** ⟦*procId*⟧

## <a name="remarks"></a>Commenti

L'argomento *procId* è valido solo in MASM a 32 bit.
