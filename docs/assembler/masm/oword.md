# <a name="oword"></a>OWORD

Utilizzato come identificatore di tipo quando è richiesto un tipo di dati a 16 byte.

## <a name="syntax"></a>Sintassi

> ⟦*nome*⟧ **OWORD**  *inizializzatore* ⟦__,__ *inizializzatore* ... ⟧
