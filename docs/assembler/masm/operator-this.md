# <a name="operator-this"></a>Operatore THIS

Restituisce un operando di *tipo* specificato i cui valori di offset e segmento sono uguali al valore del contatore di posizione corrente.

## <a name="syntax"></a>Sintassi

> **Questo** *tipo*
