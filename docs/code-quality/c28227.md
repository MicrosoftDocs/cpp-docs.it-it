# <a name="c28227"></a>C28227

> avviso C28227: l'annotazione non può essere anche un PrimOp (vedere la dichiarazione precedente)

Questo avviso indica che l'annotazione denominata viene dichiarata come un'annotazione ordinaria e che in precedenza è stata dichiarata come PrimOp. Questa operazione non dovrebbe essere possibile se le macro standard vengono utilizzate per le annotazioni. Questo avviso non dovrebbe essere visualizzato in uso tipico.
