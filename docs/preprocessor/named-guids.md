# <a name="named_guids-import-attribute"></a>named_guids attributo Import

**Sezione specifica C++**

Indica al compilatore di definire e inizializzare le variabili GUID nello stile precedente, nel formato `LIBID_MyLib` , `CLSID_MyCoClass` , `IID_MyInterface` e `DIID_MyDispInterface` .

## <a name="syntax"></a>Sintassi

> **#import** **named_guids** della *libreria di tipi*
