# <a name="push_macro-pragma"></a>Pragma push_macro

Salva il valore della macro con *Nome macro* nella parte superiore dello stack per questa macro.

## <a name="syntax"></a>Sintassi

> **#pragma push_macro ("** _Nome-macro_ **")**
