# `_bstr_t::length`

**Specifico di Microsoft**

Restituisce il numero di caratteri in `_bstr_t`, escluso il valore null finale, del `BSTR` incapsulato.

## <a name="syntax"></a>Sintassi

```cpp
unsigned int length ( ) const throw( );
```
