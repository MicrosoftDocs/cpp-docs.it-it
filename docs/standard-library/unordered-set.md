# <a name="ltunordered_setgt"></a>&lt;unordered_set&gt;

## <a name="requirements"></a>Requisiti

**Intestazione:**\<unordered_set>

**Spazio dei nomi:** std

> [!NOTE]
> La \<unordered_set> libreria usa anche l' `#include <initializer_list>` istruzione.
