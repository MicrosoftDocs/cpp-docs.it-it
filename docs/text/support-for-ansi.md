# <a name="support-for-ansi"></a>Supporto per ANSI

La maggior parte dei metodi e delle classi MFC supporta il set di caratteri ANSI, sebbene il framework MFC nel suo complesso stia evolvendo gradualmente verso il supporto solo del set di caratteri Unicode. A causa dei miglioramenti continui nei controlli comuni di Windows Vista e Windows versione 6,1, il supporto per diverse classi e metodi ANSI Ã¨ deprecato.  Per altre informazioni, vedere [API ANSI deprecate](../mfc/deprecated-ansi-apis.md) e [supporto per Unicode](../text/support-for-unicode.md).
