# <a name="compiler-error-c2134"></a>Errore del compilatore C2134

' Function ': la chiamata non genera un'espressione costante

Una funzione dichiarata come constExpr pu√≤ chiamare solo altre funzioni dichiarate come constExpr.

L'esempio seguente genera l'C2134:

```cpp
// C2134.cpp
// compile with: /c
int A() {
    return 42;
};

constexpr int B() {
    return A();  // Error C2134: 'A': call does not result in a constant expression.
}
```

Possibile soluzione:

```cpp
// C2134b.cpp
constexpr int A() {  // add constexpr to A, since it meets the requirements of constexpr.
    return 42;
};

constexpr int B() {
    return A();  // No error
}
```
