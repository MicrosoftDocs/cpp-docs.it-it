# <a name="compiler-error-c2047"></a>Errore del compilatore C2047

default non valido

La parola chiave **`default`** pu√≤ essere visualizzata solo in un' **`switch`** istruzione.

L'esempio seguente genera l'errore C2047:

```cpp
// C2047.cpp
int main() {
   int i = 0;
   default:   // C2047
   switch(i) {
      case 0:
      break;
   }
}
```

Possibile soluzione:

```cpp
// C2047b.cpp
int main() {
   int i = 0;
   switch(i) {
      case 0:
      break;
      default:
      break;
   }
}
```
