# <a name="linker-tools-error-lnk1332"></a>Errore degli strumenti del linker LNK1332

sono stati rilevati \<count> Windows Runtime tipi importati in un modulo e definiti in un altro modulo

Quando ha prodotto la destinazione corrente, il linker ha rilevato <`count`> Windows Runtime tipi, ognuno dei quali viene importato in un modulo e definito anche in un altro modulo.

### <a name="to-correct-this-error"></a>Per correggere l'errore

- Correggere tutti gli errori LNK2039 nella compilazione a seconda del suggerimento nel messaggio di errore.
