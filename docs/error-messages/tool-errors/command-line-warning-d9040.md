# <a name="command-line-warning-d9040"></a>Avviso della riga di comando D9040

> l'opzione '/analyze ' verrà ignorata; Gli avvisi di analisi del codice non sono disponibili in questa edizione del compilatore

L' **`/analyze`** opzione della riga di comando non è disponibile in tutte le edizioni di Visual Studio. Per porre rimedio a questo avviso, passare ad una versione supportata di Visual Studio o rimuovere l'opzione della riga di comando.
