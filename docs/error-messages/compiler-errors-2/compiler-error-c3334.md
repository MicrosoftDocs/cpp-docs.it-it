# <a name="compiler-error-c3334"></a>Errore del compilatore C3334

impossibile eseguire #import su una libreria dei tipi danneggiata

Il file di libreria in un'istruzione `#import` è danneggiato e non può essere utilizzato dal compilatore.  Il danneggiamento può derivare dalla generazione errata del modulo, dal danneggiamento del disco o nel caso in cui il file binario sia stato in qualche modo modificato.

Per ulteriori informazioni, vedere [#import direttiva](../../preprocessor/hash-import-directive-cpp.md).
