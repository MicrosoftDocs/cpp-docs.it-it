# <a name="compiler-error-c3812"></a>Errore del compilatore C3812

' Property ' deve essere il primo token in una dichiarazione di proprietà

Quando si dichiara una proprietà, la `__property` parola chiave deve essere il primo token sulla riga.

C3812 è raggiungibile solo usando l'opzione del compilatore obsolete **/CLR: oldSyntax**.
