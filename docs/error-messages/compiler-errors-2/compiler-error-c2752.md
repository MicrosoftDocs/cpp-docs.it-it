# <a name="compiler-error-c2752"></a>Errore del compilatore C2752

' template ': pi√π specializzazioni parziali corrispondenti all'elenco di argomenti del modello

Creazione di un'istanza ambigua.

L'esempio seguente genera l'C2752:

```cpp
// C2752.cpp
template<class T, class U>
struct A {};

template<class T, class U>
struct A<T*, U> {};

template<class T, class U>
struct A<T,U*> {};

// try the following line instead
// template<class T, class U> struct A<T*,U*> {};

int main() {
   A<char*,int*> a;   // C2752 an instantiation

   // OK
   A<char*,int> a1;
   A<char,int*> a2;
   A<char,int> a3;
}
```
