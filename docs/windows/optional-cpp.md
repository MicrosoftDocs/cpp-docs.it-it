# <a name="optional-c"></a>optional (C++)
Specifica un parametro facoltativo per una funzione membro.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
[optional]  
  
```  
  
## <a name="remarks"></a>Note  
 Il **facoltativo** attributo C++ ha la stessa funzionalit√† come il [facoltativo](http://msdn.microsoft.com/library/windows/desktop/aa367132) attributo MIDL.  
  
## <a name="example"></a>Esempio  
 Il codice seguente viene illustrato come **facoltativo** potrebbero essere utilizzati:  
  
```  
// cpp_attr_ref_optional.cpp  
// compile with: /LD  
#include "unknwn.h"  
[module(name="ATLFIRELib")];  
  
[dispinterface, uuid("00000000-0000-0000-0000-000000000001")]  
__interface IFireTabCtrl : IDispatch  
{  
   [id(1)] long procedure ([in, optional] VARIANT i);  
};  
```
