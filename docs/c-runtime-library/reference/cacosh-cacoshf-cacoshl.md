# <a name="cacosh-cacoshf-cacoshl"></a>cacosh, cacoshf, cacoshl

Ottiene il coseno iperbolico inverso di un numero complesso con discontinuità della funzione corrispondente ai valori minori di 1 lungo l'asse reale. .

## <a name="syntax"></a>Sintassi

```C
_Dcomplex cacosh(
   _Dcomplex z
);
_Fcomplex cacosh(
   _Fcomplex z
);  // C++ only
_Lcomplex cacosh(
   _Lcomplex z
);  // C++ only
_Fcomplex cacoshf(
   _Fcomplex z
);
_Lcomplex cacoshl(
   _Lcomplex z
);
```

### <a name="parameters"></a>Parametri

*z*<br/>
Numero complesso che rappresenta un angolo, espresso in radianti.

## <a name="return-value"></a>Valore restituito

Coseno iperbolico inverso di *z*, in radianti. Il risultato è non associato e non negativo lungo l'asse reale e nell'intervallo [-Iπ, + Iπ] lungo l'asse immaginario.

## <a name="remarks"></a>Commenti

Poiché C++ consente l'overload, è possibile chiamare overload di **cacosh** che accettano e restituiscono valori **_Fcomplex** e **_Lcomplex** . In un programma C **cacosh** accetta e restituisce sempre un valore **_Dcomplex** .
