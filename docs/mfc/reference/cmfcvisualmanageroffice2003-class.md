## <a name="cmfcvisualmanageroffice2003setdefaultwinxpcolors"></a><a name="setdefaultwinxpcolors"></a> CMFCVisualManagerOffice2003:: SetDefaultWinXPColors

Specifica se il gestore di visualizzazione deve utilizzare colori o colori del tema Windows XP nativi ottenuti da [GetSysColor](/windows/win32/api/winuser/nf-winuser-getsyscolor).

```
static void SetDefaultWinXPColors(BOOL bDefaultWinXPColors = TRUE);
```

### <a name="parameters"></a>Parametri

*bDefaultWinXPColors*<br/>
in Specifica se il gestore visuale utilizzerà i colori nativi di Windows XP.

### <a name="remarks"></a>Commenti

Se *bDefaultWinXPColors* è true, il gestore visuale utilizzerà colori nativi di Windows XP, ad esempio Blue, olive o Silver. In caso contrario, il gestore visuale utilizzerà i colori ottenuti da `GetSysColor` . Il gestore di visualizzazione usa elementi visivi, ad esempio COLOR_3DFACE, COLOR_3DSHADOW, COLOR_3DHIGHLIGHT, COLOR_3DDKSHADOW e COLOR_3DLIGHT.

Per impostazione predefinita, un `CMFCVisualManagerOffice2003` oggetto usa i colori del tema di Windows XP nativi.
