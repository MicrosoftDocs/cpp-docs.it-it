# <a name="cmfcvisualmanageroffice2003-class"></a>Classe CMFCVisualManagerOffice2003


##  <a name="setdefaultwinxpcolors"></a>CMFCVisualManagerOffice2003::SetDefaultWinXPColors  
 Specifica se il gestore visualizzazione debba utilizzare colori del tema native Windows XP o colori ottenuto da [GetSysColor](http://msdn.microsoft.com/library/windows/desktop/ms724371).  
  
```  
static void SetDefaultWinXPColors(BOOL bDefaultWinXPColors = TRUE);
```  
  
### <a name="parameters"></a>Parametri  
 [in] `bDefaultWinXPColors`  
 Specifica se il gestore visualizzazione utilizzerà native colori di Windows XP.  
  
### <a name="remarks"></a>Note  
 Se `bDefaultWinXPColors` è `TRUE`, il gestore visualizzazione utilizzerà native colori di Windows XP, ad esempio blu o d'argento. In caso contrario, il gestore visualizzazione utilizzerà i colori ottenuti da `GetSysColor`. Il gestore visualizzazione utilizza gli elementi visivi, ad esempio `COLOR_3DFACE`, `COLOR_3DSHADOW`, `COLOR_3DHIGHLIGHT`, `COLOR_3DDKSHADOW`, e `COLOR_3DLIGHT`.  
  
 Per impostazione predefinita, un `CMFCVisualManagerOffice2003` oggetto utilizza native colori del tema Windows XP.
