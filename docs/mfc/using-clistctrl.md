# <a name="using-clistctrl"></a>Utilizzo di CListCtrl

Utilizzare un controllo elenco per visualizzare qualsiasi disposizione di icone con etichette, come in Esplora file, oppure elenchi in formato colonna di testo, con o senza icone. Per una descrizione delle quattro possibili "viste" (da non confondere con le viste MFC), è possibile disporre di un controllo elenco, visualizzazione icone, visualizzazione icona piccola, visualizzazione elenco e visualizzazione report, vedere Visualizzazioni nella panoramica della classe [CListCtrl](../mfc/reference/clistctrl-class.md) .

In alcune visualizzazioni, gli utenti possono trascinare le icone in posizioni diverse o modificare le etichette delle icone. Come esempio vedere il riquadro destro di Esplora file, che utilizza un controllo elenco in una finestra non di dialogo. È possibile sperimentare le visualizzazioni disponibili nel menu Visualizza di Esplora risorse.

Per informazioni correlate, vedere [informazioni sui controlli List-View](/windows/win32/Controls/list-view-controls-overview) nel Windows SDK.

> [!NOTE]
> Il Windows SDK si riferisce ai controlli elenco come "controlli visualizzazione elenco". Questo utilizzo di "visualizzazione" non si riferisce alle classi di visualizzazione MFC, in particolare `CListView`. Per altre informazioni, vedere [elenco di controllo e visualizzazione elenco](../mfc/list-control-and-list-view.md).
